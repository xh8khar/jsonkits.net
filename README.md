# JSONKits — 250+ JSON Tools

A comprehensive, free, browser-based JSON toolkit built with Next.js. 100% client-side processing — no data ever leaves your browser.

## Features

- **250+ JSON tools** covering formatting, validation, conversion, viewing, comparison, generation, compression, security, extraction, and more
- **50+ converter tools** — JSON to/from YAML, CSV, XML, TOML, INI, Properties, TSV, Excel, HTML, Markdown, Plain Text
- **20 programming languages** — Generate code from JSON: TypeScript, Python, Go, Java, C#, Kotlin, Swift, Dart, Rust, Scala, Ruby, Perl, Lua, R, MATLAB, Haskell, Elm, OCaml, Zig, PHP
- **Database SQL generators** — PostgreSQL, MySQL, SQLite, SQL Server, Oracle, MongoDB, Firestore
- **API tools** — Swagger, OpenAPI, Postman, GraphQL, cURL
- **Formatting & Validation** — Formatter, Beautifier, Minifier, Validator, Linter, Repair, Syntax Checker
- **Viewing & Exploration** — Tree Viewer, Explorer, URL Viewer, File Viewer, Live Preview, Search
- **Comparison & Editing** — Diff Checker, Merge, Split, Sort, Filter, Rename, Remove Duplicates
- **Security & Compression** — Encrypt, Decrypt, Mask Data, Escape, Deflate, Gzip
- **Browser-based** — Upload, Download, Copy, Clipboard Viewer, URL Fetch

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Build

```bash
npm run build
```
